Frequently asked questions

The quick version - the long version is on how it works.

What is Is It Vibe Coded?

Is It Vibe Coded is a free, tongue-in-cheek tool that gives any website or GitHub repository a “vibe coded” score - a playful guess at whether it was written by a human or generated by AI. Paste a URL and you get an instant, shareable verdict.

Is it free?

Yes. Checking any URL is completely free and needs no sign-up. The paid “plans” on the pricing page are part of the joke - nothing can actually be purchased.

How accurate is the vibe score?

It isn’t - and that’s the point. Is It Vibe Coded is a satirical project. The score is computed deterministically from the domain name, so the same URL always returns the same verdict. There is no real code analysis.

Can I check a GitHub repository?

Yes. Paste any GitHub repo or user URL - for example github.com/facebook/react - and the tool pulls its real language, star count and description before assigning a vibe.

Do you store the URLs I check?

No. Nothing is stored. The verdict is generated from the URL and immediately forgotten, like a dream you had on a train.
